		JavaScript
Linguagem de programação interpretada, de alto nível e multiparadigma.
Comentários com // ou /* */.


	Saída de dados
let variavel = prompt("Digite:");   | Pede um valor ao usuário e guarda.

let variável = confirm("Confirme"); | Pergunta com 'OK' e 'Cancelar'.

console.log("Printa texto");	    | Exibe texto no console.

window.alert("Alerta"); 	    | Exibe a Janela de Alerta com o texto quando entra no site.
