		JavaScript
// e /* */ | Comentários.
Terminar linha com ';'.
Pode ficar dentro do HTML, usando a tag script. Ex: 
| <script> alert("Olá mundo"); </script> | Mostra caixa de alerta quando carrega.


	Variáveis
Guardar valores na memória. Ex:
| let idade = 18;   | Guarda um numero inteiro.
| let nome = "João";| Guarda texto.


	Tipos de dados
Inteiro(int): | let x = 10;
Texto (string): | let nome = "Maria";
Booleano (verdadeiro/true ou falso/false): | let ligado = true;


	Entrada e saída de dados
| alert("Bem-vindo"); | Mostra uma mensagem na tela.

| let nome = prompt("Nome: "); | Pede um valor ao usuário.

| let resposta = confirm("Continuar?"); | Pergunta com 'OK' e 'Cancelar'.


	Condicional if / else
if (idade >= 18) { 		| Se a condição se cumprir
   alert("Pode entrar");
} else {			| Senão
   alert("Entrada proibida"); }


	Operadores lógicos
Maior que: >
Menor que: <
Igual: 	   ==
Diferente: !=
E (AND):   && | Ambas condições tem de ser true.
Ou (OR):   || | Uma condição precisa ser verdadeira.


	Funções
Função é um bloco de código que pode ser reutilizado. Pode passar parâmetros.

function mostrarMensagem() {
    alert("Olá!");
}

Para executar a função:
mostrarMensagem();


	Eventos
Eventos acontecem quando o usuário interage com a página.

| <button onclick="clicou()">Clique aqui</button>
  <script> function clicou() { alert("Botão clicado"); } </script>


	DOM
DOM é a forma que o JavaScript usa para acessar o HTML.

Exemplo HTML: | <p id="texto">Olá</p> |

Exemplo JavaScript: 
| let p = document.getElementById("texto");
  p.innerText = "Texto alterado"; |

Isso muda o texto do parágrafo.

	Mudando estilo com JavaScript
| <p id="caixa">Texto</p> <script> let caixa = document.getElementById("caixa"); caixa.style.color = "red"; caixa.style.backgroundColor = "yellow"; </script> |
