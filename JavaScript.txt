Comentários com // ou /* */.


	Saída de dados
const CONSTANTE = prompt("Digite:"); | Pede um valor ao usuário e guarda. Constantes em UpperCase.

let variavel = confirm("Confirme");  | Pergunta com 'OK' e 'Cancelar'.

console.log("Printa texto");	     | Exibe texto no console.

window.alert("Alerta"); 	     | Exibe a Janela de Alerta com o texto quando entra no site.


	Eventos
Eventos acontecem quando o usuário interage com a página.

| <button onclick="clicou()">Clique aqui</button>
  <script> function clicou() { alert("Botão clicado"); } </script>


	DOM
DOM é a forma que o JavaScript usa para acessar o HTML.

document.getElementBy...('identificador'); | Pega o elemento com esse Id/ClassName/Name/TagName.

document.getElementById('p1').textContent = "P alterado com textContent."; | Altera o texto do <p> de id '.p1'


	Conversão de dados
let n = window.prompt("Digite um numero: "); | Pensa que é string.
n = Number(n) ou Boolean(n) ou String(n);    | Transforma variável nesse tipo.
