		Array
let frutas = ["Banana", "Maçã", "Laranja"];

frutas.push("x")      | Adiciona o elemento no FINAL do array. Retorna o novo tamanho.
frutas.pop()          | Remove o último elemento do array. Retorna o elemento removido.
frutas.shift()        | Remove o primeiro elemento do array. Retorna o elemento removido.
frutas.unshift(2)     | Adiciona elemento no início do array. Retorna o novo tamanho.
frutas.map(x=>x*2)    | Cria novo array aplicando a função em cada elemento (não altera o original).
frutas.filter(x=>x>1) | Cria novo array com elementos que passam na condição (true).

let novaFruta = [... frutas, "Pera"]; | Cria novo array com elementos de Frutas + "Pera".
let novoNome  = [... "Jonas"];        | Cria array com cada letra do nome como elemento.


		Função
function pessoa(nome, idade) { return console.log(`Ola, ${nome}, de ${idade} anos.`); }
Pode não ter parâmetros, mas não terá argumentos. return nesse caso opcional. Quando tem, a função para quando chegar nele.

function Rest(...frutas) { console.log(frutas) } | Recebe quantidade variável de argumentos, Printa o array de argumentos.
Rest("Uva", "Melancia", "Laranja"); 		 | Passa 3 argumentos, que serão um array dentro da função.


		Spread Operator
let Spread = [1, 2, 3];
console.log(...Spread); | Printa separadamente, sem colchetes ou vírgulas.


		Callback
Função passada como argumento para outra função, para ser executada depois.
