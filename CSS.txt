			CSS
Cascading Style Sheets é uma linguagem de estilização usada para definir a aparência de páginas web, controlando cores, layouts, fontes e espaçamentos.


	Modificações possíveis
background-color      | Cor de fundo.
color 		      | Cor do texto.
text-align  	      | Alinha o texto (left, center, right, justify).
text-decoration       | Adiciona sublinhado, risco, etc.
text-transform        | Transforma texto em maiúsculo/minúsculo.
line-height 	      | Espaçamento entre linhas.
width 		      | Largura do elemento.
height 		      | Altura do elemento.
min/max-height/widht  | Altura minima/maxima que um elemeto pode/deve ter.
top/left/right/bottom | Distância em relação ao extremo desse lado.
outline 	      | Contorno fora da borda.
display: flow-root;   | Img não fica fora da borda do body.
columns 	      | Divide conteúdo em colunas.
vertical-align 	      | Alinha verticalmente (usado em tabelas).
clear:both/left/right | Impedi elementos flutuarem ao lado de elementos que usaram float.
box-sizing:border-box | Faz width ser tam final,impede q padding e borda aumente o elemento.
margin/direcao/:auto  | Contorno externo do elemento,em direção especifica ou auto centraliza
padding 	      | Espaço interno entre elemento e a borda.
float: direcao	      | Permite elemento ao lado de imagem sem ela ocupar toda linha.
cursor: xx;	      | Vide os vários tipos de cursores.

font-family/size/weight/style | Define fonte/tamanho do texto/espessura.


	Medidas
Absolutas: pt, px, pc, in, cm, mm. Não mudam dependendo do contexto.
  Px: mais usada. Para bordas, sombras e pequenos ajustes.
  Cm, mm e in: Quase nunca usadas. Apenas impressão (print CSS).

Relativas à Fonte: em, rem, ex, ch. Mudam dependendo da fonte.
  rem: Relativo ao tamanho da fonte do html. Se html tem 16px, 2em=32px. Fontes, padd,margin.

Relativas à Tela(Viewport): vw, vh, %, dvh, dvw, svh, lvh. Baseadas no tamanho da tela.
  vw, vh: 1vw/h = 1% da largura/altura da tela.
  dvh, dvw, svh, lvh: Criados para corrigir problemas de barra do navegador em celular.

Relativas ao Container: %, cqw, cqh, cqi, cqb. Baseado no tamanho do elemento pai.

	Combinators
Explicam a relação entre seletores. Dizem quem afeta quem.

| #container p   | Todo p que está em dentro de container.
| #container > p | Apenas os p que estão imediatamente dentro de container e ñ +profundo.
| #container ~ p | Todo p que seja irmão do container e venha depois dele.
| #container + p | O primeiro p imediatamente após o container.

	Display
Propriedade especifica se/como um elemento é mostrado.
Block-level: Começa em uma linha nova e ocupa tudo, ex: h1, div, p, form, header, footer.
Inline: Não começa numa linha nova e ocupa o necessário, ex: span, a, img.

#block{
    background-color:blue;
    width: 10%;            | Tem q especificar tamanho pra nn ocupar toda linha.
    display: block; 	   | Transforma em block-level, se já ñ fosse.
    display: inline; 	   | Transforma em inline.
    display: none; } 	   | Fica invisível e ñ ocupa espaço.
#inline{
    background-color: #3ddd45;
    width: 10%; 	      | Ñ funciona pois inline ocupa apenas o necessário.
    display: inline-block;    | Mistura, ocupa apenas o necessário do tamanho especificado.
    height: 50px;
    visibility: hidden; }     | Fica invisível.

	Pseudo-classes e pseudo-elementos
Pseudo-classe modifica um elemento quando em certo estado, Pseudo-elemento modifica algo em específico. Pseudo-classe com ':'. Pseudo-elemento com '::'
A ordem de precedência importa, .dropdown:hover .conteudo != .dropdown .conteudo:hover

:not(:hover/active/etc) | Enquanto não estiver nesta condição.

button:hover{comando;} | Passar mouse em cima.
      :active 	       | Estiver sendo clicado.

a:link{}	       | Modifica um link antes de clicar.
 :visited	       | Quando já visitado.

li:nth-child(2n+1){}   | Seleciona elemento pela posição q ocupam dentro do pai.even,odd,2...

#container p{ display: none; }	       |
#container:hover p{ display: inline; } | Fazem um texto invisivel aparecer quando houver.

h1/p...::first-letter{} | Altera primeira letra.
       ::first-line{}	| Primeira linha.
       ::selection{}    | Quando estiver com o texto selecionado.
       
h1/p...::before{content: "emoji, texto,etc";} | Antes do conteudo ser exibido, adiciona algo.
       ::after{content: "🥬";}				  | Na frente do conteúdo.

li::marker{ content: "📝"; }  				  | Em listas, altera o marcador dos <li>.


	Paginação
Conjunto de links que levam para páginas diferentes e destacam a página atual.

 <a href="pagina2.html"> 2 </a>       | Quando fazer a proxima pagina, altera paginatual.
 <a href="pagina3.html"> 3 </a>
 <a href="pagina2.html"> > </a>       | Muda quando estiver na outra pagina.
</div>

.paginacao  { text-align: center; }

    text-decoration: none; 		| Tira sublinhado de links.
    margin: 20px; } 	   		| Distância entre os links.

| "Acesse div classe paginacao, depois o link a classe "paginatual". |
.paginacao a.paginatual{ 
    background-color: green; font-weight: bolder; border-radius: 20%; }

| Modifique quando passar em cima menos o paginatual. |
.paginacao a:hover:not(.paginatual) { background-color: #222217; border-radius: 20%; }

	Flexbox
Uma maneira de organizar o layout. Unidimencional: escolhe entre linha ou coluna.

display: flex;         	      | Transforma em Flexible Box, permite modificações de flexbox.
flex-direction: row;   	      | row-reverse(ao contrario), column(coluna), column-reverse.
flex-wrap: wrap/wrap-reverse; | Se em tela menor ficarem apertados, desce a linha.
flex-flow: row wrap; 	      | Shorthand de direction e wrap.
flex-basis: 200px; 	      | O tamanho inicial do item flex no eixo principal.
flex-shrink: 1; 	      | Se pode encolher.
flex-grow: 0;   	      | Se pode crescer, 0 = não.
justify-content: center;      | flex-end, space-around/between. Muda horizontalmente.
gap: 20px; 		      | Espaço entre os itens.
align-items: center; 	      | Alinha itens, flex-end e flex-start.
align-content: center; 	      | Com flex-wrap. Diz como se comporta ao quebrar linha.
align-self: flex-end; 	      | Alinha apenas um especifico, se não tivesse wrap.
order: 1; 		      | Ocupa o último lugar. -1: ocupa o primeiro.

<div class="container">
 <div class="boxs" id="box1"> 1 </div>
 <div class="boxs" id="box2"> 2 </div>
 <div class="boxs" id="box3"> 3 </div>
 <div class="boxs" id="box4"> 4 </div> </div>

.containers{ display: flex; flex-direction: row; justify-content: center; gap: 20px; 
     column-gap:3em; height:30vh; align-items:center; flex-wrap:wrap; align-content:center; }

.boxs{ width: Xpx; height: Xpx; color: black; font-size: Xem; text-align: center; 
       border-radius: Xpx; flex-basis: 200px; flex-grow: 0; flex-shrink: 1; }
#boxs1{ background-color: red; align-self: flex-end; order: 1; }
#boxs2{ background-color: green; order: -1; }

rotateX(180deg);      	  | Rotaciona de cima pra baixo.
rotateZ(45deg);       	  | Gira.
rotate3d(1, 1, 0, 45deg); | Rotaciona nos três.
skew(10deg, 10deg);       | Distorce.
scale(1, 0.5);            | Estica ou comprime.

      transform: translate(50px, -8px) rotateY(180deg) rotateX(180deg) rotateZ(45deg) scale(1, 0.4) skew(10deg, 10deg); }

		Grid
Grades, uma alternativa de organização do layout ao flexbox. Importante envolver em container.
Usar a ferramenta do grid view no 'inspecionar elemento'.

grid container : Container que envolve o grid, display:grid nele.
grid content   : O conteúdo completo, cada bloco dentro do container.
grid item      : O item dentro de cada grid content.
grid track     : Fileira(row track) ou coluna(column) de grid itens.
grid lines     : Linhas entre os itens. Cada linha tem um número de 1 a n, ou o último pode ser -1 a -n.
grid area      : Conjunto de itens próximos.

: repeat(X, auto); | Valores repetidos podem ser resumidos, (3, 1fr) = 1fr 1fr 1fr.
: 1fr 2fr 1fr;	   | Medida fracionária, divide em frações a tela. 1fr 2fr(dobro da anterior).
space-		   | Controlam como o espaço extra é distribuído entre linhas e colunas dentro do container.
: space-between;   | Primeiro item encosta no início, último item encosta no final.
: space-around;    | Cada item recebe espaço nas duas laterais, entre itens é maior que nas extremidades.
: space-evenly;    | Distribui o espaço igualmente entre tudo, inclusive bordas.
span x; 	   | Quantos grid lines além da inicial deve espandir até o fim

	Propriedades apenas para Grid-container
display: grid;       		    | Possibilita usar o layout grid.
gap: Xpx;	     		    | Altera apenas o espaçamento entre os itens(grid lines).
grid-template-columns: auto auto... | Separa conteúdo em n colunas, de acordo com o número de parâmetros.
grid-template-rows: Xpx Xpx Xpx Xpx | Separa conteúdo em n linhas, Xpx cada.
align-items: start; center; end     | Alinhamento vertical de grid items. 'stretch' ocupa todo espaço.
justify-items: start; center; end   | Alinhamento horizontal dos grid items.
align-content: space-		    | Alinhamento vertical dos grid content. start, center e end tmb.
justify-content: space-		    | Alinhamento horizontal dos grid content.
place-items: start center;	    | Shorthand para align/justify-items.
place-content: space- end	    | Shorthand para align/justify-content.
grid-template: Xpx Xpx / 1fr 1fr    | Shorthand para grid-template-rows/columns.
grid-auto-rows: 100px		    | Novas linhas criadas ou elementos deslocados terão este tamanho.
grid-auto-columns: 100px	    | Novas colunas criadas ou elementos deslocados terão este tamanho.
grid-auto-flow: row/column;	    | Novos elementos serão inseridos em nova linha ou coluna.

	Propriedades para Grid-items
grid-row-start: X;    	      | Define o início da altura de um grid-content.
grid-row-end: X/-X/span X;    | Define o fim da altura de um grid-content.
grid-column-start: X; 	      | Define o início da largura de um grid-content.
grid-column-end: X/-X/span X; | Define o fim da largura de um grid-content.
grid-row: X/x;		      | Shorthand para grid-row-start/end.
grid-column: X/x;	      | Shorthand para grid-column-start-end.
grid-area: rs cs re ce;	      | Shorthand para grid-row-column.


		Media Queries e Responsividade
Adaptar o layout de acordo com a tela do usuário. Celulares, tablets, etc.
Optar sempre por usar %, vh, vw em width, height, img e container ao invés de px.
Usar rem ao invés de px para textos.

<link rel="stylesheet" href="print.css" media="print">  | Criar CSS responsivo só para impressora, se tira links e botões.

@media (orientation: landscape) { ... } 		| Quando tela deitada nos dispositivos móveis.

@media (min-width: 100px) and (max-width: 320px){ ... } | Se adapta ao tamanho da tela do usuário.
